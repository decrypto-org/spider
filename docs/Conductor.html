Class: Conductor

Conductor(cutOffDepth, torPort)

The conductor is the one controlling the spidering through the web. Therefor it needs to access the network module and the models/database.
A note about the HTTP Status Codes: We use this field to indicate further, non-standard errors or obstacles. This enables a leaner database design and can easily be filtered out when analyzing the data.
  • -1 indicates that the access to the given url was forbidden by robots.txt policy. We follow the policies given in the robots file
